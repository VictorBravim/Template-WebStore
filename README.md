## <code>Introdução</code>

Este projeto é um aplicativo web baseado em Next.js para uma loja ficticia chamada "WebStore". Projeto feito com intuito de praticar front-end.

## <code>Pré-requisitos</code>

- Node.js (version 14 or higher)
- npm (Node Package Manager)
 
## <code>Configuração</code>

1. Clone este repositório:

```
git clone https://github.com/VictorBravim/WebStore.git
```

2. Navegue até o diretório do projeto:

```
cd WebStore
```

3. Instale dependências:

```
npm install
or
yarn install
```

4. Execute o servidor de desenvolvimento:

```
npm run dev
```

## <code>Estrutura</code>

- component/: Contém componentes React usados ​​em todo o aplicativo.
- pages/: contém componentes de página Next.js que representam diferentes rotas do aplicativo.
- public/: contém ativos estáticos, como imagens.
- styles/: Contém estilos CSS globais, bem como estilos específicos de componentes.
- gitignore: Especifica arquivos não rastreados intencionalmente para serem ignorados.
- next.config.js: arquivo de configuração Next.js.
- package.json: arquivo de manifesto para npm, inclui metadados e dependências do projeto.
- README.md: Contém documentação do projeto e instruções para desenvolvedores.
- tsconfig.json: arquivo de configuração TypeScript.
